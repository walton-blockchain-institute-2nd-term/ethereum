# 우분투 설치

    1. 가상머신(VM)을 설치한다.
    2. 디스크 이미지 파일(iso)를 다운 받는다.
    3. 가상머신에 디스크 이미지로 운영체제를 받는다.


#  가상머신을 설치한다.
## 가상머신이란?
- 컴퓨터 안에 소프트웨어를 통해 하나의 컴퓨터를 더 만드는 기술
- 하드웨어를 소프트웨어적으로 구현해서 그 위에서 운영체제가 작동할 수 있도록 하는 기술(출처 : 생활코딩 )
